# Alessa

Objetivo:
Jogo educativo de caça-palavras.

Tecnologias:
- React
- TypeScript
- Vite

Persistência:
- localStorage

Entidades:
- Player
- MatchRecord

Regras importantes:
- Não alterar lógica de pontuação.
- Não alterar sistema de palavras.
- Não alterar ranking local.
- Não alterar fluxo de jogo.

Nova funcionalidade:
- Enviar resultados para Google Apps Script.
- Manter localStorage funcionando.
- Falha na internet não pode quebrar o jogo.
